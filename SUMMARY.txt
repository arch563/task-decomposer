items without cluttering your code. Tasks are stored in SQLite and displayed as
virtual text (like LSP diagnostics) at their code locations.

  â€¢ Persistent across sessions (SQLite storage)
  â€¢ Visual feedback without code modification (virtual text)
  â€¢ Jump from tasks to code locations
  â€¢ Filter and organize tasks in tree view
